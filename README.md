# NCF Ambiental

Site institucional da NCF Ambiental desenvolvido com Next.js, React e Tailwind CSS.

## Executar localmente

Requer Node.js 20.9 ou superior.

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000`.

## Validar a aplicação

```bash
npm run lint
npm run build
```

## Principais páginas

- Serviços residenciais e locação de lixeiras
- Coleta de resíduos volumosos
- Serviços comerciais
- Resíduos industriais e contêineres Roll-On
- Soluções comerciais por setor
- Consulta de área de atendimento por CEP
- Sobre a NCF Ambiental
